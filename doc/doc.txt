Для запроса адреса используйте форму записи следующего типа:
<устройство> <функция Modbus> <группа> <канал/параметр>

Разберем каждую составляющую по отдельности:

Устройства:

БАЗИС-12 =  "b12"   - !на данном этапе не реализовано!
БАЗИС-14 =  "b14"   - реализованы только 2, 3, 4 функции
БАЗИС-21 =  "b21"   - реализованы только 2, 3, 4 функции
БАЗИС-100 = "b100"  - !на данном этапе не реализовано!
БАЗИС-ПВ =  "bpv"   - !на данном этапе не реализовано!

Функции Modbus:

readCoil              = "f1"
readDiscreteInput     = "f2"
readHoldingRegister   = "f3"
readInputRegister     = "f4"
writeSingleCoil       = "f5"
writeSingleRegister   = "f6"
writeMultipleRegister = "f16"

Группа и канал/параметр:

Зависит от устройства и функции, конкретнее ниже.

Если устройство БАЗИС-12...

Если устройство БАЗИС-14:
    - Для 2 функции (02 (0x02) Read Discrete Inputs):
        список груп:
            "I"  - собственные аналоговые/NAMUR входные каналы (каналы с 1 по 8) 
            "F1" - кнопка Ф1 (канал 1, вводим 1)
            "F2" - кнопка Ф2 (канал 1, вводим 1)
		    "DI" - собственные дискретные входные каналы (каналы с 1 по 8)
            "EI" - входные каналы на шине расширения (каналы c 1 по 12)
            "B"  - внешние каналы (каналы с 1 по 8)
		    "P"  - расчетные каналы (каналы с 1 по 16)
            "W"  - выходные каналы (каналы с 1 по 64)

    - Для 3 функции (03 (0x03) Read Holding Registers):
        список групп:
            "CL<номер контура 1..8>" (Control Loop, контур регулирования) имеет следующие параметры:

                "ctrlMod"       - режим основного контура регулирования
                "ctrlConfig"    - конфигурация основного контура регулирования
                "coefGroup"     - номер активной группы коэффициентов основного контура регулирования
                "specAlgNum"    - номер специального алгоритма основногоконтура регулирования
                "setpoint"      - значение задания основного контура регулирования
                "valveValue"    - Значение клапана основного контура регулирования
                "Ko"            - Коэффициент Ко основного контура регулирования
                "Kp"            - Коэффициент Кп основного контура регулирования
                "Ti"            - Коэффициент Ти основного контура регулирования
                "Td"            - Коэффициент Тд основного контура регулирования
                "Tf"            - Коэффициент Тф основного контура регулирования
                "specKo"        - Коэффициент Ко спец. алгоритма регулирования контура регулирования
                "specKp"        - Коэффициент Кп спец. алгоритма регулирования контура регулирования
                "specTi"        - Коэффициент Ти спец. алгоритма регулирования контура регулирования
                "specD1"        - Настройка D1 спец. алгоритма регулирования контура регулирования
                "specD2"        - Настройка D2 спец. алгоритма регулирования контура регулирования

            "TIME" (время) имеет следующие параметры:
                "YEAR"  - год
                "MONTH" - месяц
		        "DAY"   - день
                "HOUR"  - час
		        "MIN"   - минута
                "SEC"   - секунда

            "P" - расчетные каналы (каналы с 1 по 24) !НЕ РЕАЛИЗОВАНО! 

    - Для 4 функции (04 (0x04) Read Input Registers):
        список групп:
            "I"     - входные каналы (канал с 1 по 8)
            "EXT"   - канал шины расширения (канал с 1 по 8)
            "P"     - расчетные каналы (канал с 1 по 16)
            "B"     - внешние каналы (канал с 1 по 8)
                        

Если устройство БАЗИС-21:
    - Для 2 функции (02 (0x02) Read Discrete Inputs):
        список групп:
            "I1"-"I3"                              - входные каналы (канал с 1 по 16)
		    "I4"                                   - входные каналы (канал с 1 по 8)
            "I11"-"I15", "I21"-"I23", "I31"-"I33"  - входные каналы (канал с 1 по 8)
            "P"                                    - расчетные каналы (канал с 1 по 24)
            "B"                                    - внешние каналы (канал с 1 по 128)
            "W1"                                   - выходные каналы (канал с 1 по 5)
            "W2"-"W4"                              - выходные каналы (канал с 1 по 10)
            "W5"-"W9"                              - выходные каналы (канал с 1 по 20)

    - Для 3 функции (03 (0x03) Read Holding Registers):
        список групп:
            CLM (Control Loop Main, основной контур регулирования) имеет следующие параметры:

                "ctrlMod"       - режим основного контура регулирования
                "ctrlConfig"    - конфигурация основного контура регулирования
                "coefGroup"     - номер активной группы коэффициентов основного контура регулирования
                "specAlgNum"    - номер специального алгоритма основногоконтура регулирования
                "setpoint"      - значение задания основного контура регулирования
                "valveValue"    - Значение клапана основного контура регулирования
                "Ko"            - Коэффициент Ко основного контура регулирования
                "Kp"            - Коэффициент Кп основного контура регулирования
                "Ti"            - Коэффициент Ти основного контура регулирования
                "Td"            - Коэффициент Тд основного контура регулирования
                "Tf"            - Коэффициент Тф основного контура регулирования
                "specq"         - Настройка q спец. алгоритма
                "specD"         - Настройка D спец. алгоритма

            CLEXT (Control Loop External, внешний контур регулирования) такие же параметры, но нет "ctrlMod", "specq", "specD"

            TIME (время) имеет следующие параметры:
                "YEAR"  - год
                "MONTH" - месяц
                "DAY"   - день
                "HOUR"  - час
                "MIN"   - минута
                "SEC"   - секунда

            "P" - расчетные каналы (каналы с 1 по 24) !НЕ РЕАЛИЗОВАНО! 

    - Для 4 функции (04 (0x04) Read Input Registers):
            список групп:
                "I1", "I2", "I3" - входные каналы с (каналы с 1 по 16)
                "I4" - входной канал (каналы с 1 по 8)
                "I11"-"I15", "I21"-"I23", "I31"-"I33" - входные каналы (каналы с 1 по 12),
                "P" - расчетные каналы (каналы с 1 по 24)
                "B" - внешние каналы (каналы с 1 по 128)
                "V1", "V2" - аналоговые выходные каналы (каналы с 1 по 8)
                "HI1"-"HI3", "HI11"-"HI15", "HP" - часовые значения с входных каналов (канал с 1 по 8) и с расчетных каналов (канал с 1 по 24)
                параметр вводится следующим образом: <канал><день><час>,
                        где канал - соответственно, номер канала,
                            день:
                                t (today)                   - сегодня,
                                y (yesterday)               - вчера,
                                b (day before yesterday)    - позавчера;
                            час - соответственно, час :)
                            
                            Пример: HI3 8y20 - группа 3, канал 8, вчера за 20 час
